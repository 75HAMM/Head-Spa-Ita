🚀 COME USARE TUTTO ORA
═══════════════════════════════════════════════════════════════════════════

**STEP 1: SCARICA TUTTI I FILE** (5 minuti)

   📥 Tutti i file si trovano in: /mnt/user-data/outputs/
   
   Scarica:
   • index.html
   • chi_siamo_head_spa_italia.html
   • contatti_head_spa_italia.html
   • area_riservata_operatori.html
   • blog_head_spa_articoli_3.html
   • README.md
   • .gitignore
   • Le 4 guide Markdown

   Salva in una cartella locale (es: ~/Desktop/head-spa-italia/)


**STEP 2: CREA REPOSITORY SU GITHUB** (2 minuti)

   1. Vai a: https://github.com/new
   2. Repository name: head-spa-italia
   3. Public: ✅
   4. Add README: ✅
   5. Create repository
   6. COPIA L'URL (es: https://github.com/username/head-spa-italia.git)


**STEP 3: SETUP GIT LOCALE** (3 minuti)

   Apri terminale e scrivi:

   ```bash
   # Configura Git
   git config --global user.name "Il Tuo Nome"
   git config --global user.email "tua.email@example.com"

   # Clona repository
   cd ~/Desktop
   git clone https://github.com/TUO_USERNAME/head-spa-italia.git
   cd head-spa-italia

   # Crea cartelle
   mkdir html
   mkdir assets
   ```


**STEP 4: COPIA FILE** (2 minuti)

   ```bash
   # Copia i file scaricati nella cartella

   cp ~/Downloads/index.html .
   cp ~/Downloads/chi_siamo_head_spa_italia.html html/chi-siamo.html
   cp ~/Downloads/contatti_head_spa_italia.html html/contatti.html
   cp ~/Downloads/area_riservata_operatori.html html/area-riservata.html
   cp ~/Downloads/blog_head_spa_articoli_3.html html/blog.html
   cp ~/Downloads/README.md .
   cp ~/Downloads/.gitignore .
   ```


**STEP 5: UPLOAD SU GITHUB** (2 minuti)

   ```bash
   # Aggiungi file
   git add .

   # Commit
   git commit -m "Initial commit: Head Spa Italia sito completo"

   # Upload
   git push -u origin main
   ```

   ✅ Il tuo sito è ora su GitHub!


**STEP 6: TEST LOCALHOST** (1 minuto)

   ```bash
   # Dalla cartella head-spa-italia
   python3 -m http.server 8000

   # Apri browser: http://localhost:8000
   ```

   ✅ Visualizza il sito localmente!


**STEP 7: METTI ONLINE GRATIS** (3 minuti)

   1. GitHub → repository head-spa-italia
   2. Settings → GitHub Pages
   3. Branch: main
   4. Salva
   5. Attendi 1-2 minuti
   6. Accedi a: https://username.github.io/head-spa-italia

   ✅ Sito online e pubblico!


⏱️ TEMPO TOTALE PER SETUP: 15-20 MINUTI

📞 SUPPORTO E AIUTO
═══════════════════════════════════════════════════════════════════════════

Se trovi problemi:

**Git non trovato:**
   Scarica da: https://git-scm.com

**Username/Password GitHub:**
   Crea Personal Access Token: https://github.com/settings/tokens

**GitHub Pages non si aggiorna:**
   • Attendi 1-2 minuti
   • Svuota cache browser (Ctrl+Shift+Del)
   • Verifica branch: Settings → Pages

**Localhost non funziona:**
   • Assicurati di essere nella cartella giusta
   • Verifica: python3 --version
   • Prova porta diversa: python3 -m http.server 8001

**Help Online:**
   • Git Docs: https://git-scm.com/book
   • GitHub Docs: https://docs.github.com
   • MDN: https://developer.mozilla.org
   • Stack Overflow: https://stackoverflow.com
