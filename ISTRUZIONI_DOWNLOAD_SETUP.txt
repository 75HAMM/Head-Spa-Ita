**PASSO 1: CREA CARTELLE LOCALI** (1 minuto)
────────────────────────────────────────────────

Apri File Manager e crea questa struttura:

~/Desktop/
└── head-spa-italia/          ← Cartella principale
    ├── index.html            ← Sarai qui
    ├── README.md
    ├── .gitignore
    └── html/                 ← Sottocartella
        ├── chi-siamo.html
        ├── contatti.html
        ├── area-riservata.html
        └── blog.html

👉 Comandi per creare cartelle:
```bash
mkdir -p ~/Desktop/head-spa-italia/html
cd ~/Desktop/head-spa-italia
```


**PASSO 2: SCARICA I FILE** (3 minuti)
────────────────────────────────────────────────

Scarica da: /mnt/user-data/outputs/

I file sono già pronti. Copiali dove indicato:

1. index.html → ~/Desktop/head-spa-italia/
2. chi_siamo_head_spa_italia.html → ~/Desktop/head-spa-Italia/html/ (rinomina in chi-siamo.html)
3. contatti_head_spa_italia.html → ~/Desktop/head-spa-italia/html/ (rinomina in contatti.html)
4. area_riservata_operatori.html → ~/Desktop/head-spa-italia/html/ (rinomina in area-riservata.html)
5. blog_head_spa_articoli_3.html → ~/Desktop/head-spa-italia/html/ (rinomina in blog.html)
6. README.md → ~/Desktop/head-spa-italia/
7. .gitignore → ~/Desktop/head-spa-italia/


**PASSO 3: TEST LOCALHOST** (2 minuti)
────────────────────────────────────────────────

Apri terminale:

```bash
cd ~/Desktop/head-spa-italia
python3 -m http.server 8000
```

Apri browser: http://localhost:8000

✅ Se vedi la HOME PAGE → OK!


**PASSO 4: CREA REPOSITORY GITHUB** (2 minuti)
────────────────────────────────────────────────

1. Vai a: https://github.com/new
2. Repository name: head-spa-italia
3. Public: ✅
4. Add README: ✅
5. Clicca: Create repository
6. COPIA L'URL che vedi (es: https://github.com/tuousername/head-spa-italia.git)


**PASSO 5: SETUP GIT** (2 minuti)
────────────────────────────────────────────────

Apri terminale e scrivi:

```bash
# Configura Git (una sola volta)
git config --global user.name "Your Full Name"
git config --global user.email "your.email@gmail.com"

# Verifica configurazione
git config --global --list
```


**PASSO 6: CLONA REPOSITORY** (2 minuti)
────────────────────────────────────────────────

```bash
# Naviga in Desktop
cd ~/Desktop

# Clona il repository
git clone https://github.com/TUO_USERNAME/head-spa-italia.git

# Entra nella cartella
cd head-spa-italia
```


**PASSO 7: COPIA FILE NEL REPO** (1 minuto)
────────────────────────────────────────────────

Se hai scaricato in ~/Downloads:

```bash
# Crea cartelle
mkdir html

# Copia file
cp ~/Downloads/index.html .
cp ~/Downloads/chi_siamo_head_spa_italia.html html/chi-siamo.html
cp ~/Downloads/contatti_head_spa_italia.html html/contatti.html
cp ~/Downloads/area_riservata_operatori.html html/area-riservata.html
cp ~/Downloads/blog_head_spa_articoli_3.html html/blog.html
cp ~/Downloads/README.md .
cp ~/Downloads/.gitignore .
```


**PASSO 8: VERIFICA STRUTTURA** (1 minuto)
────────────────────────────────────────────────

Verifica che i file siano nel posto giusto:

```bash
# Visualizza struttura
ls -la
tree  # o find . -type f

# Output atteso:
# .gitignore
# README.md
# index.html
# html/chi-siamo.html
# html/contatti.html
# html/area-riservata.html
# html/blog.html
```


**PASSO 9: AGGIUNGI A GIT** (1 minuto)
────────────────────────────────────────────────

```bash
# Verifica cosa aggiungerai
git status

# Aggiungi tutti i file
git add .

# Verifica nuovamente
git status
```


**PASSO 10: PRIMO COMMIT** (1 minuto)
────────────────────────────────────────────────

```bash
git commit -m "Initial commit: Head Spa Italia complete setup with 5 HTML pages"
```


**PASSO 11: CARICA SU GITHUB** (2 minuti)
────────────────────────────────────────────────

```bash
git push -u origin main
```

Se chiede username/password:
- Username: il tuo username GitHub
- Password: il tuo Personal Access Token (vedi sotto)

👉 Come creare Personal Access Token:
1. Vai: https://github.com/settings/tokens
2. Clicca: Generate new token
3. Nome: head-spa-italia-push
4. Seleziona: repo (checkbox)
5. Clicca: Generate
6. COPIA il token
7. Usalo come password in terminale


**PASSO 12: ATTIVA GITHUB PAGES** (3 minuti)
────────────────────────────────────────────────

1. Vai su GitHub → repository head-spa-italia
2. Clicca: Settings (ingranaggio in alto a destra)
3. Scorri fino a: "GitHub Pages"
4. Seleziona:
   - Branch: main
   - Folder: (root)
5. Clicca: Save
6. Attendi 1-2 minuti
7. Accedi a: https://tuousername.github.io/head-spa-italia

✅ Il tuo sito è online!
