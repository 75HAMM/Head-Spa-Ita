✨ STEP PRINCIPALE (COPIA-INCOLLA)
═══════════════════════════════════════════════════════════════════════════

Se hai fretta, copia-incolla questi comandi:

// 1. Configura Git (una volta sola)
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

// 2. Crea cartelle
mkdir -p ~/Desktop/head-spa-italia/html
cd ~/Desktop/head-spa-italia

// 3. Copia i file HTML scaricati
cp ~/Downloads/index.html .
cp ~/Downloads/chi_siamo_head_spa_italia.html html/chi-siamo.html
cp ~/Downloads/contatti_head_spa_italia.html html/contatti.html
cp ~/Downloads/area_riservata_operatori.html html/area-riservata.html
cp ~/Downloads/blog_head_spa_articoli_3.html html/blog.html
cp ~/Downloads/README.md .
cp ~/Downloads/.gitignore .

// 4. Test localhost
python3 -m http.server 8000
// Apri: http://localhost:8000

// 5. GitHub setup
// Vai a: https://github.com/new
// Crea repository "head-spa-italia"
// Copia il link
git clone [TUO_LINK]
cd head-spa-italia

// 6. Upload
git add .
git commit -m "Initial commit: Head Spa Italia setup"
git push -u origin main

// 7. GitHub Pages
// GitHub → Settings → Pages → Branch main → Save
// Attendi 1-2 minuti
// Vai a: https://username.github.io/head-spa-italia

FATTO! ✅

🆘 SE TROVI ERRORI
═══════════════════════════════════════════════════════════════════════════

PROBLEMA: "fatal: not a git repository"
SOLUZIONE: cd ~/Desktop/head-spa-italia
           Assicurati di essere nella cartella giusta

PROBLEMA: "git: command not found"
SOLUZIONE: Installa Git da https://git-scm.com

PROBLEMA: "Permission denied" GitHub
SOLUZIONE: Crea Personal Access Token: https://github.com/settings/tokens

PROBLEMA: GitHub Pages non si aggiorna
SOLUZIONE: Attendi 1-2 minuti, svuota cache browser (Ctrl+Shift+Del)

PROBLEMA: Localhost non funziona
SOLUZIONE: python3 --version  (verifica Python installato)
           Prova porta diversa: python3 -m http.server 8001
